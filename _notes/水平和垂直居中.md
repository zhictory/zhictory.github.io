---
title: "水平和垂直居中"
---

```
/* 水平居中 */

div {
  width: 750px;
  margin: 0 auto;
}

/* 完全居中 */

div {
  position: absolute;
  top: 0;
  bottom: 0;
  left: 0;
  right: 0;
  margin: auto;
  width: 100px;
  height: 100px;
}

div {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}
```
