---
title: "Truthy 和 Falsy"
---

Truthy

```
if (true)
if ({})
if ([])
if (42)
if ("0")
if (new Date())
if (-42)
if (3.14)
if (-3.14)
if (Infinity)
if (-Infinity)
```

Falsy

```
if (false)
if (null)
if (undefined)
if (0)
if (0n)
if (NaN)
if ('')
if ("")
if (``)
```
